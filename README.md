🤖 RAG Telegram Bot

Telegram-бот, который отвечает на вопросы по твоим документам с помощью локальной ML-модели.

---

📌 Описание

Проект реализует подход RAG (Retrieval-Augmented Generation):

1. Загружает документы (TXT, PDF)
2. Разбивает их на части
3. Создаёт векторную базу
4. Находит релевантные фрагменты
5. Генерирует ответ через локальную модель

---

🚀 Возможности

- 📂 Поддержка ".txt" и ".pdf"
- 🔍 Поиск по содержимому документов
- 🧠 Генерация ответов через LLM
- 🤖 Telegram-интерфейс
- 💻 Полностью локальная работа (через Ollama)

---

🛠️ Технологии

- Python
- LangChain
- ChromaDB
- Sentence Transformers
- Ollama (LLaMA3)
- python-telegram-bot

---

📁 Структура проекта

project/
├── bot.py          # Telegram бот
├── rag.py          # Логика RAG
├── data/           # Документы
├── db/             # Векторная база
└── venv/           # Виртуальное окружение

---

⚙️ Установка

1. Клонировать репозиторий

git clone https://github.com/USERNAME/RAG-Tg-bot.git
cd RAG-Tg-bot

2. Создать окружение

python3 -m venv venv
source venv/bin/activate

3. Установить зависимости

pip install -r requirements.txt

---

📥 Подготовка данных

Помести файлы в папку "data/":

data/
├── test.txt
└── file.pdf

---

🧠 Запуск модели

Установи и запусти Ollama:

ollama run llama3

---

🏗️ Построение базы

python

from rag import build_db
build_db()

---

🤖 Запуск бота

1. Создай бота через BotFather
2. Вставь токен в "bot.py"

Запуск:

python bot.py

---

💬 Примеры запросов

- Что такое машинное обучение?
- Объясни RAG
- О чем этот документ?

---

⚠️ Важно

- Перед запуском бота обязательно:
  - запустить Ollama
  - построить базу ("build_db()")
